# 📊 Monthly Refund Report

A beautiful, self-contained HTML dashboard for tracking monthly plugin refunds.

| Month | Total Refunds | Link |

| May 2026 | 12 | [View →](https://mdibrahimk48.github.io/refund-report/) |

## How to Add a New Month's Report

1. Duplicate `index.html` and rename it (e.g., `june-2026.html`)
2. Update the data in the new file
3. Add a row to the table above in `README.md`
4. Commit and push — GitHub Pages auto-deploys

## Tech Stack

- Pure HTML + CSS (no build step required)
- [Chart.js](https://www.chartjs.org/) for interactive charts
- Hosted via [GitHub Pages](https://pages.github.com/)
